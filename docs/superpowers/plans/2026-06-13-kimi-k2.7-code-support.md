# Add `kimi-k2.7-code` support to the `kimi-coding` provider — Implementation Plan

**Goal:** Update the built-in `kimi-coding` provider preset so that `kimi-k2.7-code` becomes the default model while `kimi-for-coding` remains selectable.

**Architecture:** A single data change in `presets.go` drives the new default for both Claude Code and Codex (Codex already copies the Claude preset via `codexKimiCodingPreset()` in `agent.go`). A small regression test in `presets_test.go` locks in the default model and model list.

**Tech Stack:** Go, standard `testing` package.

### Task 1: Add a regression test for the `kimi-coding` default model

**Files:**
- Create: `presets_test.go`
- Test: `presets_test.go`

- [ ] **Step 1: Write the failing test**

Create `presets_test.go` at the repository root with the following content. It asserts that the `kimi-coding` preset defaults to `kimi-k2.7-code` and still offers `kimi-for-coding`.

```go
package main

import (
	"slices"
	"testing"
)

func TestKimiCodingDefaultModel(t *testing.T) {
	preset, ok := providerPresets["kimi-coding"]
	if !ok {
		t.Fatal("kimi-coding preset not found")
	}

	if preset.Model != "kimi-k2.7-code" {
		t.Errorf("default model = %q, want %q", preset.Model, "kimi-k2.7-code")
	}

	wantModels := []string{"kimi-k2.7-code", "kimi-for-coding"}
	for _, want := range wantModels {
		if !slices.Contains(preset.Models, want) {
			t.Errorf("models %v does not contain %q", preset.Models, want)
		}
	}

	for _, field := range []struct {
		name string
		got  string
	}{
		{"Haiku", preset.Haiku},
		{"Sonnet", preset.Sonnet},
		{"Opus", preset.Opus},
		{"Subagent", preset.Subagent},
	} {
		if field.got != "kimi-k2.7-code" {
			t.Errorf("%s = %q, want %q", field.name, field.got, "kimi-k2.7-code")
		}
	}
}
```
